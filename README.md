This is a fix/remake of the original Guaranteed Start.

Any advice let me know!

Features:
- Replace the item ONLY in the first Treasure Room of the run *you go in* (that means u can skip for planetarium) with a 3 or more quality item.
- It works with Greed Mode :D (read below)
- Now it works with Mod Config Menu Pure!!
- It works with mods using the corresponding pools.
- I'm pretty sure it only chooses items the player has unlocked.
- If you are playing The Lost, it gives you birthright effect for the new item.
- It's my first mod :D

About Mod Config Menu

Now you can change how the mod works!!!
Since the original Mod Confid Menu Pure seems to be deleted from Steam, you can either download the repository on Github: https://github.com/Zamiell/isaac-mod-config-menu
Or installing a reuploaded version.

(last news: no version is working LMAO, so follow the next instructions:)

If you have problems with any of these options you can opt to go to this mod folder at "C:\Program Files (x86)\Steam\steamapps\common\The Binding of Isaac Rebirth\mods" (on Windows) and change directly the "config.lua" file, and change FGS.config, not the default config. And then reload the mod.

Features.
- You can choose which rooms should be affected by the mod.
- Also change some item preferences!
- And can spawn a little portal to the treasure room at the start of the run

About Greed Mode
- It now works for the greed silver room, and the greed treasure room, but the moment you enter to one of those, it won't affect the other one. So you can choose to enter the silver room, or wait for a better item in the treasure room (recommended in my experience).
(You can select which room should be affected in the mod config)
Note: The Greed Boss Pool has very few quality 3 items, so u should see the same items very often.
Note 2: I made minimum Q4 config so it would NOT work for Silver Room (Greed Boss Pool) in Greed Mode, because the only Q4 item is Magic Mushroom lol, so it's a little boring, pls go to the treasure room instead :D

Known Bug (?):
- If you re-roll the item, results in Glowing Hourglass, and then use it; the previous chosen item by the mod will not appear, instead, it will be the (probably) original item of the seed.
